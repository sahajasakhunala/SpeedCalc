User Manual


OVERVIEW
SpeedCalc is not just a calculator.
It is a reaction-based math battle game designed to improve mental math speed, accuracy, and focus under pressure.

The game challenges you to solve arithmetic problems against the clock while managing lives, streaks, and escalating difficulty. As you improve, the game adapts to push you harder.


WHAT SpeedCalc IS

SpeedCalc is a terminal-based math game that combines:
-Fast-paced arithmetic challenges
-Time pressure
-Streak-based progression
-Boss battles
-A shield (life) system
-Leaderboard tracking
The game runs on Windows, macOS, and Linux.


STARTING THE GAME

When the game launches, you will be guided through the setup automatically.
You will be asked to select a game mode before beginning.

GAME MODES
1) Classic Mode:
Classic Mode is designed for learning and warming up.

-Each question allows 60 seconds
-The time limit never decreases
-Mistakes consume shields
-Ideal for beginners or relaxed play

2) Hyper-Speed Mode:
Hyper-Speed Mode is designed for experienced players.

-Starts with 60 seconds
-Time limit decreases as your streak increases
-Minimum time per question is 15 second
-No pause after wrong answer
-Focuses on reaction speed and consistency


HOW A ROUND WORKS

At the start of each round, you are given options:
a) Start a math challenge
b) Quit the game and view the leaderboard
Choosing to start will immediately present a math problem.


ANSWERING QUESTIONS
Each question is displayed clearly on screen.

Important rules:
-Do not type early
-Input is only accepted when prompted
-Type your answer carefully
-Submit your answer once you are finished
Typing before input is enabled may cause incorrect behavior.


DIVISION RULES (Very Important)
Division problems require exact formatting.
-Answers must be rounded to exactly two decimal places
-Even whole-number results must include two decimals
If the format is incorrect, the answer is marked wrong even if the value is correct.

CORRECT ANSWERS
When you answer correctly:
-A confirmation message is shown
-A high-pitched sound plays
-Your streak increases by one
Maintaining streaks is key to progression.


WRONG ANSWERS
When you answer incorrectly:
-An error message is shown
-A low-pitched sound plays
-A shield may absorb the mistake
The game continues as long as you have shields.


TIMEOUTS
If you fail to answer before time runs out:
-The question counts as incorrect
-Shields are affected the same way as a wrong answer


SHIELD SYSTEM
Shields act as extra lives.
-You start with 1 shield
-Maximum shields: 3

When You Make a Mistake
-If you have shields remaining, one shield is consumed
-If you have no shields left:
  -Your streak resets to zero
  -Your score may be recorded on the leaderboard

SHIELD REWARDS
-Every 5 correct answers in a row, you earn one shield
-Shields are only awarded if you are not already at maximum
Consistent performance is rewarded.


BOSS LEVELS
Every 25 correct answers in a row triggers a boss challenge.
Boss challenges include:
-Larger numbers
-Harder operations
-Increased difficulty and pressure

Defeating a Boss
-Instantly restores shields to maximum (3)
Boss levels act as milestones in your run.


LEADERBOARD
-Tracks the top 5 highest streaks
-Scores are saved automatically
-Leaderboard loads every time the game starts
-Displayed when you quit the game
Your best runs are always preserved.
